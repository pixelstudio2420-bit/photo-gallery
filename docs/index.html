⚡ Quick Start

เร็วสุดจาก zero ถึง deployed ใน 3 คำสั่ง

1
เช่า VPS
Ubuntu 22.04 อย่างน้อย 2GB RAM
2
SSH เข้าเครื่อง
รัน provision.sh
3
เสร็จใน ~10 นาที
Script ทำทุกอย่างให้
# On fresh Ubuntu 22.04 VPS as root:
curl -fsSL https://raw.githubusercontent.com/YOUR/REPO/main/docs/deployment/provision.sh -o provision.sh
sudo bash provision.sh
# Answer 6-7 questions → wait ~10 min → site is live 🎉

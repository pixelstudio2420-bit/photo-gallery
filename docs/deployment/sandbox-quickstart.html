🎯 เป้าหมาย

คู่มือนี้ deploy Postgres fork (ที่อยู่ใน C:\xampp\htdocs\photo-gallery-pgsql) ขึ้น Laravel Cloud Sandbox tier (ฟรี) เพื่อทดสอบก่อนจ่าย Production tier. Sandbox มี $30 credit ฟรีตอนสมัครใหม่ — พอใช้รัน serverless Postgres + 1 compute instance นาน ~30 วัน

1Push fork ขึ้น GitHub (1 นาที)

PS> cd C:\xampp\htdocs\photo-gallery-pgsql
PS> git init
PS> git add .
PS> git commit -m "initial: pgsql fork"
PS> git branch -M main
PS> git remote add origin git@github.com:YOU/jabphap-pgsql.git
PS> git push -u origin main
⚠️ ตรวจ .gitignore มี .env + vendor/ + node_modules/ ก่อน push

2สมัคร Cloud + connect GitHub (3 นาที)

  1. เข้า cloud.laravel.com/register → Sign up with GitHub
  2. Verify email + ใส่บัตรเครดิต (จะไม่ตัดถ้าใช้ใน free credit)
  3. Dashboard → Source Providers → GitHub → Connect
  4. Grant access เฉพาะ repo jabphap-pgsql

3สร้าง app + Postgres + Redis (3 นาที)

3.1 New Application

  1. คลิก + New Application
  2. From GitHub → YOU/jabphap-pgsql · branch main
  3. Region: Singapore (ap-southeast-1)
  4. Tier: Sandbox
  5. Click Create

3.2 Add Serverless Postgres ⭐

  1. tab Databases → + New Database
  2. Type: Serverless Postgres ← จุดเด่นของ fork นี้
  3. Name: jabphap-pg
  4. Compute: 0.25 CU (auto-scale)
  5. Click Create — ใช้เวลา ~30 วินาที
💡 Auto-wired ENV: Cloud ใส่ DB_HOST, DB_PORT=5432, DB_DATABASE, DB_USERNAME, DB_PASSWORD, DB_SSLMODE=require ให้เอง — ไม่ต้องมี .env

3.3 Add Valkey (Redis-compatible)

  1. tab Caches → + New Cache
  2. Type: Valkey · Size: 128MB
  3. Auto-wires REDIS_HOST + REDIS_PASSWORD

4ตั้ง ENV ที่ขาด (2 นาที)

DB + Redis auto-wired แล้ว — เหลือแค่ ENV ที่ Cloud ไม่รู้:

APP_NAME="จับภาพ"
APP_ENV=production
APP_KEY=# กด "Generate" ใน Cloud UI
APP_DEBUG=false
APP_URL=https://jabphap-pgsql.laravel.cloud
APP_TIMEZONE=Asia/Bangkok
DB_CONNECTION=pgsql     # ← สำคัญ! บอกให้ใช้ pg driver

SESSION_DRIVER=redis
CACHE_STORE=redis
QUEUE_CONNECTION=redis

FILESYSTEM_DISK=r2
R2_ACCESS_KEY_ID=YOUR_KEY
R2_SECRET_ACCESS_KEY=YOUR_SECRET
R2_BUCKET=jabphap-photos
R2_ENDPOINT=https://abc.r2.cloudflarestorage.com

LINE / Stripe / Omise / Google ENV ทั้งหมดเหมือนคู่มือ main Cloud guide step 7

💰 Monitor cost

Sandbox + free credit = $0 สำหรับเดือนแรก แต่ดูได้ว่ามี usage เท่าไหร่:

  1. Dashboard → Billing → Current Usage
  2. ดู:
    • Compute hours used (ฟรี ~720 hrs/month ใน Sandbox)
    • Postgres CU-hours (ใน Sandbox = pay-per-use แต่ scale-to-zero ตอนไม่มี traffic = เกือบฟรี)
    • Bandwidth (100GB/mo ฟรี)
  3. ตั้ง Spending Limit = $10/month เพื่อกัน surprise
